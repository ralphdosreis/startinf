# 💻 Frontend - Chatbot StartINF

## 📄 Descrição

Este diretório contém a interface de usuário do **Chatbot StartINF**.  
A interface é responsável por permitir que estudantes e administradores interajam com o sistema por meio de uma aplicação web moderna, focada em usabilidade e agilidade no suporte ao ecossistema de startups.

---

## 🎯 Responsabilidades

- **Interface de chat:** Prover um ambiente amigável para interação em tempo real com o assistente de IA.
- **Visualização de respostas e artefatos:** Renderização clara de textos, documentos e materiais gerados pelo sistema para o usuário.
- **Interface administrativa:** Área dedicada para a gestão de documentos, editais e curadoria de conhecimento.
- **Comunicação com o backend:** Gerenciamento de chamadas de API, estados de carregamento e persistência de dados da sessão.

---

## 🛠 Tecnologias

As seguintes ferramentas foram escolhidas para garantir performance e escalabilidade:

- **React:** Biblioteca principal para construção da interface baseada em componentes.
- **Vite:** Ferramenta de build de última geração para um desenvolvimento rápido e otimizado.
- **CSS:** Estilização customizada para garantir a identidade visual do projeto.

## 🛠️ Como rodar os projetos localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/Residencia-em-TIC-Turma-2/S02-Apoema.git
    ```
2.  Entre na pasta do projeto desejado:
    ```bash
    cd S02-Apoema
    ```
3.  Vá para a branch dev:
    ```bash
    git checkout dev
    ```
4.  Entre na pasta do Frontend:
    ```bash
    cd .\frontend\
    ```
5.  Instale as dependências:
    ```bash
    npm install
    ```
6.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
