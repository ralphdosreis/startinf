# 💻 Frontend - Chatbot StartINF

## 🛠️ Como rodar os projetos localmente

1.  Clone o repositório:
    ```bash
    git clone https://github.com/ralphdosreis/startinf.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
